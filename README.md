# Athlete-Injury-Prediction
What is this?
This is a machine learning model trained on a data set that contains information regarding football players, their age and injury and other statsticaly analyses.

Through this project I learned:
The differences between unsupervised and supervised learning for a model.
How to check the trained model for overfitting and how to correct the model if it is overfitting based on the sample data.
I learned how to clean up the data for the model so that it is easier for the code to digest. For example converting categories into true and false data sets for the model, as well as dropping useless colummns that don't contribte to the training of the model for the specific goal, in this case that was injury prediction. Also learning to readjust scales so that bigger numbers don't necessarily mean drastic effects, but keeping in mind the ranges of the data provided for the model.
I also learned hwo to orgainize a testing set for the model so that it can be checked after it was done being trained, to facilitate the final checks for the model.

How to run:
You would want to upload the data sheet that contains the data sheet that you want to train the model on into the files portion on to your google colab. Make sure it is in the outermost layer/folder so that the program can find the data file correctly.
Then you just need to run the poritons of code block by block into the google colab notebook and make sure that each block is running before proceeding to the next block.
